Test problem I and II consists of square meshes with $N_b=256$ boundary nodes and $N=4225$ nodes, while problem III uses an ``L''-shaped mesh with $N_b=288$ boundary nodes and $N=6127$ nodes.


\subsection{Computational setup}
All numerical experiments were performed on a laptop running Fedora Linux 43 with Linux kernel version 6.19. The system was running on an Intel(R) Core(TM) i7-10510U (4 cores, 8 threads) with 16GB of memory.

Python 3.9 was used for all computations, together with NumPy and SciPy for linear algebra routines \cite{harrisArrayProgrammingNumPy2020,virtanenSciPy10Fundamental2020}. The legacy FEniCS software was used for solving PDEs using the finite element method \cite{alnaesFEniCSProjectVersion2015,loggAutomatedSolutionDifferential2012}. All library specifications, algorithm implementations, and notebooks used to generate the results in this thesis are openly available at: \href{https://github.com/EliasSev/masters-thesis}{github.com/EliasSev/masters-thesis}.
